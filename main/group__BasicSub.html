 Q7 vector subtraction.
 

Description

Element-by-element subtraction of two vectors.

    pDst[n] = pSrcA[n] - pSrcB[n],   0 <= n < blockSize.

There are separate functions for floating-point, Q7, Q15, and Q31 data types.
